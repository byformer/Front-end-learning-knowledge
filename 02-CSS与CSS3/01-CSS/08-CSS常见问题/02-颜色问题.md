# 颜色问题

> 深入理解 CSS 六种颜色模式：https://xiaohuochai.site/CSS/render/color/colorMode.html

## 关键字

- CSS 支持 17 种合法命名颜色，浏览器支持 140 种颜色

## 16 进制

- `#ffffff`

## RGB 模式

- 256 个单元：0 ~ 255，`rgb(r, g, b)`

## RGBA 模式

- 增加了透明度，`rgb(r, g, b, a)`

> IE 浏览器使用滤镜实现透明色

## HSL 模式

- `hsl(h, s, l)`

## HSLA 模式

-增加了透明度，`hsl(h, s, l, a)`
